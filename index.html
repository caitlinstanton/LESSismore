What is

LESS?

Less is a dynamic stylesheet language that can be described as a superset of CSS.
It dds variables, operations, function-like elements, and JavaScript to already existing CSS syntax, to make the creation of an aesthetic product more logical and less time-consuming.

The History of LESS

It was created in 2009 by Alexis Seller. It incorporates influences from Sass and CSS, based mainly on the syntax of those two technologies, and was originally written in Ruby. It was eventually ported to JavaScript.
In May of 2012, it was handed over to a team of contributors and developers. Its iterations have, in turn, influenced Sass, the LESS framework, and Bootstrap.

Variables

  • Defined using '@name: value'
  • Store a value so variable can be used instead of the value itself
  • Helps keep track of values easily so that they can be changed later without difficulty

Variable Scope

  • Variables can be defined inside and outside CSS rules
  • Variables can also be used to define other variables

Manipulating Variables

  • Operations can be used to change the value of variables

Nesting

  • Can write specific CSS rules within each other, so that they only apply to a certain section
  • Gets rid of the cascading style of CSS, as well as the need to rewrite classes, IDs, etc.

Example of nesting in LESS

Mixins

  • Can add properties from other CSS rules to a CSS rule, saving you typing
  • Parametric mixins: Mixins can also take in multiple variables to change how they properties work when being called in different CSS rules
  • Can be given a default; still allows you to change the parameter of the mixin

Example of mixins in LESS

Additional Features

Importing Files

  • Can import other files to use variables and mixins from those files
  • Can use reference to avoid outputting content of variables and mixins
  • Done using
    @import (reference) "file_name";

Extending

  • Used to reduce redundant code
  • Uses same rules for a different mixin or variable
  • Similar to classes and subclasses in Java
  • Done using
    &:extend(.external_class);

Merging Property

  • Used for CSS properties that accept multiple values
  • Merge values that belong to the same property
  • Add properties from another class by writing it as a rule in the general rule:
    .inner-shadow { box-shadow: inset 10px 10px 5px #ccc; }
    .box { .inner-shadow; }

Using LESS

  • Download less.js file from the LESS website
  • Insert the less.js file < script src="less.js" type="text/javascript">< /script>
  • Create a .less file and link it to your page
    <link rel="stylesheet/less" type="text/css" href="style.less">
  • For production, it is recommended to use node.js or another third-party client to compile beforehand
